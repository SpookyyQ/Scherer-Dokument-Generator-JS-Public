Einblick

Installation für Techniker

  1. Repository klonen und Abhängigkeiten installieren.
  2. Python 3 und pywin32 installieren.
  3. Windows-Drucker einrichten (Kyocera für Boardkarte/Checkliste).
git clone https://github.com/SpookyyQ/Scherer-Dokument-Generator-JS.git
cd Scherer-Dokument-Generator-JS
npm install
py -3 -m pip install pywin32
npm run start
