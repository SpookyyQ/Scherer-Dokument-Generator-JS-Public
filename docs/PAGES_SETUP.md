# GitHub Pages Setup

1. Open repository settings on GitHub.
2. Go to **Pages**.
3. Under **Build and deployment**, set **Source** to `GitHub Actions`.
4. Save.
5. Push to `main` (or run the workflow manually under **Actions**).

After a minute, your site will be available at:

`https://spookyyq.github.io/Scherer-Dokument-Generator-JS-Public/`

(If the repository name changes, the URL path changes too.)
